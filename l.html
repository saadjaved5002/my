Unordered Listing

Explanation

An unordered list in HTML is used to group a set of list items that do not have any particular order or sequence. These list items are typically represented by bullet points or similar markers.

Syntax

  • The unordered list is created using the
    <ul></ul>
    tags.
  • Each item inside the unordered list is created using the
    <li>
    (list item) tag.
Example:

  • Hockey
  • Cricket
