HTML

Description

HTML (HyperText Markup Language) is the standard language used to create and design websites. It structures content on the web by defining elements like headings, paragraphs, links, images, and multimedia. HTML is composed of various tags that provide instructions to the browser on how to display the content.






















Code example

		<!DOCTYPE html>
		<html>
		<head>
			<meta charset="utf-8">
			<meta name="viewport" content="width=device-width, initial-scale=1">
			<title> Here the page title is written </title>
		</head>
		<body>
		This is how HTML is written.
		</body>
		</html>
